# Backend

## Tech Stack

- Go v1.24
- SQLite
- Meilisearch

## Getting Started

1. Install Go v1.24 or higher
2. Navigate to the backend directory:
   ```
   cd backend
   ```
3. Install dependencies:
   ```
   go mod download
   ```
4. Run the application:
   ```
   go run main.go
   ```

## Development

- Build the application:
  ```
  go build -o bin/main
  ```
